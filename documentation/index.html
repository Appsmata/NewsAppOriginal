Info App

This Android News App latest version 1.2.6, added more amazing featured such as Activity Guide Splash, Material Ripple, News Slider Fragment, Category Fragment, Search Featured Live Viewed, Send Notification to Single and Multiple Device, Share Image and Text to Whatsapp, Facebook, Instagram and more.

 

Configure Server Php

After you upload this admin panel to your hosting, don't forget for set up or configureation with server. More detail can you see image below. Go to file powerrender\includes\variables.php

 

For active comment system please Go to folder powerrender\api\users\config.php, change config database with your database config

 

Next one, change base url image public for display image on your admin panel, Go to powerrender\component\image_public.php

 

Change firebase key, Go to folder powerrender\push.php

Increment and Decrement News Slider on Home, Go to powerrender\api\get-news-slider.php

Set up for limit news, i suggest you set > 7 , Go to folder powerrender\api\get-all-news.php

Configure Android

For display data from server, you can configure or change base url on android project, Go to folder app->java->powerrender->retrofitconfig->BaseUrlConfig.java. For details please see image below

Change ID Admob

Ads Admob display by value, so if you input 3 so ads adbom will be appear when 3 activity ready. To configure or change id admob Go to folder res->values->strings.xml

To display ads admob by values, you can Go to folder res->values->integers.xml

Change Text Guide

For change guide text title, description and image you can Go to folder res->values->strings.xml
